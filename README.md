# 🧪 비밀연구소 V3 (Genesis) 배포 가이드

대표님! 앱이 **V3로 업그레이드** 되면서 배포 방법이 아주 조금 바뀌었습니다.
(더 강력해진 만큼, **폴더**를 올려야 합니다!)

## 🚀 3초 만에 인터넷에 올리기 (Netlify Drop)

1. **빌드하기** (한 번만 하면 됩니다)
   - VS Code 터미널에서 다음 명령어를 입력하세요:
     ```bash
     npm run build
     ```
   - 잠시 기다리면 좌측 탐색기에 **`dist`** 라는 폴더가 새로 생깁니다.

2. **업로드하기**
   - 인터넷 브라우저에서 [Netlify Drop](https://app.netlify.com/drop) 사이트에 접속하세요.
   - 방금 생긴 **`dist`** 폴더를 통째로 사이트 화면에 드래그(끌어다 놓기) 하세요.
     - (주의: `비밀연구소 앱` 전체가 아니라, 그 안에 있는 `dist` 폴더만 올려야 합니다!)

3. **끝!**
   - 10초 뒤에 나오는 주소를 복사해서 사용하시면 됩니다.

## 📱 앱 설치 (PWA)
- 아이폰/갤럭시에서 "홈 화면에 추가"를 누르면 앱처럼 설치됩니다.
- V3부터는 인터넷이 끊겨도 일부 기능(기록 보기 등)이 작동합니다.
