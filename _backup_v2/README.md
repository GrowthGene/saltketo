# 🧂 솔티키토 (SaltyKeto) 배포 가이드

대표님! 이 앱을 전 세계 사람들과 공유하는 가장 쉬운 방법입니다. 💖

## 🚀 3초 만에 인터넷에 올리기 (Netlify Drop)

이 방법은 **설치도 필요 없고, 무료**입니다!

1. 인터넷 브라우저에서 [Netlify Drop](https://app.netlify.com/drop) 사이트에 접속하세요.
2. 지금 보고 계신 **`비밀연구소 앱` 폴더를 통째로** 사이트 화면에 드래그(끌어다 놓기) 하세요.
3. 끝! 10초 뒤에 `random-name.netlify.app` 같은 주소가 나옵니다.
4. 그 주소를 복사해서 카톡방이나 인스타에 올리면 됩니다!

---

## 💡 팁
- 나중에 주소를 예쁘게 바꾸고 싶으면 Netlify에 가입(무료)해서 **'Site Settings > Change site name'**에서 바꿀 수 있어요.
    - 예: `salty-keto.netlify.app`
- 앱을 업데이트하고 싶으면?
    - 수정한 폴더를 다시 똑같이 드래그하면 덮어씌워집니다!

## 📱 앱 설치처럼 쓰려면?
아이폰/갤럭시에서 사파리나 크롬으로 접속한 뒤, **"홈 화면에 추가"** 버튼을 누르면 진짜 앱 아이콘처럼 바탕화면에 생겨요!
